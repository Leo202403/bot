📊 查看状态
  bash 服务器管理.sh status

  bash 服务器管理.sh backtest           # DeepSeek
  bash 服务器管理.sh backtest qwen      # Qwen

📈 生成数据（安全模式，会临时停止AI）
  bash 服务器管理.sh export 7           # 7天数据

🛑 停止所有服务
  bash 服务器管理.sh stop

▶️  启动所有服务
  bash 服务器管理.sh start

🔄 重启所有服务
  bash 服务器管理.sh restart

🧹 清理内存
  bash 服务器管理.sh clean

⚠️  核心原则（1GB内存限制）

  ❌ 数据导出和AI服务不可同时运行
  ✅ 使用"服务器管理.sh export"会自动管理内存
  ✅ 直接运行"python3 export_historical_data.py"会导致OOM

🆘 问题排查

服务器卡死：
  bash 服务器管理.sh clean

服务异常：
  bash 服务器管理.sh restart

查看内存占用：
  free -h
  ps aux --sort=-%mem | head -6

检查进程：
  ps aux | grep python | grep -v grep

强制清理导出进程：
  pkill -9 -f export_historical_data

────────────────────────────────────────────────────────────────────────

📋 健康状态指标

✅ 正常：
  内存：< 600MB/970MB（< 62%）
  Swap：< 200MB/1024MB（< 20%）
  进程：deepseek + qwen + web = 3个

⚠️  警告：
  内存：600-800MB（62-82%）
  Swap：200-500MB（20-50%）
  建议：bash 服务器管理.sh clean

❌ 危险：
  内存：> 800MB（> 82%）
  Swap：> 500MB（> 50%）
  立即：bash 服务器管理.sh stop → clean → start
