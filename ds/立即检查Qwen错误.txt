⚠️  常见启动失败原因：

1. Python 依赖缺失
   → 执行: bash ds/安装所有依赖.sh

2. 环境变量文件损坏或缺失
   → 检查 .env.qwen 是否存在
   → 检查 API_KEY 是否正确

3. 代码语法错误
   → 查看错误日志中的 Python traceback

4. 端口冲突
   → 检查是否有其他 Qwen 进程在运行

5. 文件权限问题
   → 检查 trading_data/qwen/ 目录权限
